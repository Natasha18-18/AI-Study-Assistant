# 🎓 Final AI Study

Final AI Study is a full-stack MERN application that helps students learn smarter using AI-powered conversations, file uploads, authentication, and personalized study support.

---

# 🚀 Features

- 🤖 AI-powered study assistant
- 💬 Chat-based learning interface
- 🔐 JWT Authentication
- 📧 OTP Email Verification
- 📁 File & Image Upload Support
- ⭐ Save Important Responses
- 👤 User Profile Management
- ⚡ Fast & Responsive UI
- 🌙 Modern Clean Design

---

# 🛠️ Tech Stack

## Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcryptjs
- Nodemailer
- Multer

## AI Integration
- OpenAI API

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/final-ai-study.git
cd final-ai-study
```

---

# 🔧 Install Dependencies

## Frontend

```bash
npm install
```

## Backend

```bash
cd backend
npm install
```

---

# 🔐 Environment Variables

## Root `.env`

```env
VITE_API_URL=http://localhost:5000
```

## Backend `.env`

```env
PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key
OPENAI_API_KEY=your_openai_key

EMAIL_USER=your_email
EMAIL_PASS=your_app_password
```

---

# ▶️ Run Project

## Start Frontend

```bash
npm run dev
```

## Start Backend

```bash
cd backend
npm run dev
```
